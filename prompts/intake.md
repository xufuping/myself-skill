# 基础信息录入脚本

## 开场白

```
我来帮你创建你的电子影分身。只需要回答 5 个问题，每个都可以跳过。
```

---

## 问题序列

### Q1：花名/代号（必填）

```
你想让别人怎么称呼你的影分身？（花名、昵称、网名都行）

例：Ding / 老吴 / CodeMonkey
```

- 接受任意字符串
- slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"老吴" → `lao-wu`）
- 英文直接小写 `-` 连接（"Big Mike" → `big-mike`）

---

### Q2：基础身份

```
用一句话介绍自己——年龄、性别、所在地、职业，想到什么写什么。

例：27岁 男 深圳 前端工程师
例：95后 女 北京 自由职业 内容创作者
```

从回答中解析（缺失的留空）：
- **年龄/年龄段**
- **性别**
- **所在地**
- **职业/行业**

---

### Q3：性格标签

```
用一句话描述你的性格——MBTI、星座、生肖、性格特点，随便写。

例：INFP 双鱼座 属龙 社恐但网上话多 完美主义 拖延症晚期
例：ENTJ 狮子座 属虎 控制欲强 工作狂 但对朋友很暖
```

从回答中识别并提取（缺失的留空）：
- **MBTI**：16 种标准类型
- **星座**：12 星座
- **生肖**：12 属
- **性格标签列表**：从标签库匹配 + 接受自定义描述

#### 性格标签库

**社交维度**：社恐 / 社牛 / 话痨 / 闷骚 / 独来独往 / 社交达人 / 表面外向内心内向

**工作维度**：完美主义 / 差不多就行 / 拖延症 / 工作狂 / 效率控 / 摸鱼大师

**决策维度**：果断 / 纠结 / 理性驱动 / 感觉驱动 / 数据驱动

**情绪维度**：情绪稳定 / 玻璃心 / 表面淡定 / 容易焦虑 / 乐天派 / 阴阳怪气

**其他**：控制欲强 / 佛系 / 理想主义 / 实用主义 / 强迫症 / 三分钟热度

---

### Q4：自我认知

```
你觉得自己是个什么样的人？用你自己的话说，不需要客观。

例：表面随和但内心有很多想法不说出来，做事容易三分钟热度但真正在意的事会死磕到底
例：很理性，不喜欢废话，做什么事情都要先想清楚为什么
```

直接保留原文，不做解析。这是 Persona Layer 0 的核心输入。

---

### Q5：价值主张

```
你最擅长什么？如果别人要"租用"你的影分身，你能提供什么价值？

例：React + TypeScript 全栈开发，擅长从0到1搭建中后台系统
例：10年运营经验，擅长冷启动和用户增长策略
例：心理咨询师，擅长情绪疏导和人际关系分析
例：什么都略懂一点，生活经验丰富，擅长给建议
```

直接保留原文。这是 Knowledge Part A 的核心锚点。

---

## 确认汇总

收集完毕后展示：

```
信息汇总：

  👤  {花名}
  📍  {年龄} {性别} {所在地} {职业}（未填则省略）
  🧠  {MBTI} {星座} {生肖}（未填则省略）
  🏷️  性格：{标签列表}（未填则省略）
  💭  自我认知：{自我认知原文}（未填则省略）
  💡  价值主张：{价值主张原文}（未填则省略）

确认无误？（确认 / 修改 [字段名]）
```

用户确认后进入 Step 2 数据导入。
