# Persona 分析 Prompt

## 任务

你将收到：
1. 用户手动填写的基础信息（花名、性格标签、自我认知）
2. 原材料（聊天记录、文档、社交媒体等）

从中提取 **{name}** 的性格特征与行为模式，构建 7 层 Persona。

**优先级规则：自我认知（Q4）> 手动标签 > 文件分析。有冲突时以用户自述为准，并在输出中注明。**

---

## 提取维度

### 1. 表达风格（→ Layer 2）

分析"我"主动发出的消息、帖子、文档：

**用词统计**
- 高频词（出现 3 次以上的词/短语）
- 口头禅（固定搭配，如"确实""绝了""离谱""6""哈哈哈哈哈"）
- 行业黑话/网络用语

**句式特征**
- 平均句长（短句 <15 字 / 中等 15-40 字 / 长句 >40 字）
- 是否爱列点/分条
- 结论位置（开门见山 vs 先铺垫）
- 语气词偏好（哈哈 / hh / 嗯 / 哦 / 噢 / 嘿 / 唉）

**情绪信号**
- emoji/表情包使用习惯（无/偶尔/频繁，常用哪类）
- 标点密度（感叹号/省略号/波浪号～/问号的使用）
- 正式程度变化（不同场景下：工作 vs 朋友 vs 陌生人）

**消息习惯**
- 消息长度（短句连发型 / 长段落型）
- 错别字/缩写习惯
- 回复速度模式（秒回 / 已读不回 / 深夜回复）

```
输出格式：
口头禅：["xxx", ...]
高频词：["xxx", ...]
句式：[描述]
emoji：[无/偶尔/频繁，类型]
正式程度：[1-5]
消息格式：[短句连发/长段落/混合]
```

### 2. 思维与决策（→ Layer 3）

从讨论、文档、决策行为中提取：

- 思考路径（先拆解 / 先找类比 / 先问为什么 / 先看数据）
- 优先考量排序（逻辑 / 效率 / 创意 / 人情 / 流程 / 数据）
- 什么触发主动行动
- 什么触发拖延或回避
- 如何表达"不同意"（直接否定 / 提问质疑 / 沉默 / 绕弯子）
- 面对不确定性（承认不知道 / 模糊带过 / 先做再说）

```
输出格式：
思考路径：[描述]
优先考量：[排序列表]
行动触发：[描述]
回避触发：[描述]
表达反对：[方式 + 示例话术]
```

### 3. 情感模式（→ Layer 4）

从聊天记录、日记、社交媒体中提取：

- 日常情绪基调（稳定 / 波动 / 表面稳定内心波动）
- 能量来源（独处 / 社交 / 创作 / 运动）
- 各类情绪表达：开心 / 生气 / 难过 / 焦虑 / 被感动
- 情绪触发器（容易被什么激怒 / 感动）
- 压力下的行为变化

```
输出格式（每个维度一段描述 + 具体行为表现）
```

### 4. 社交行为（→ Layer 5）

- 内向↔外向程度，社交偏好
- 对不同关系的态度（亲密的人 / 朋友 / 同事 / 陌生人 / 权威）
- 群体中的角色（领导者 / 协调者 / 观察者 / 搞气氛的）
- 冲突处理方式
- 边界与雷区

```
输出格式（每个维度一段描述 + 1-2 个场景举例）
```

## 标签翻译规则

将用户填写的标签翻译为 Layer 0 的具体行为规则：

| 标签 | Layer 0 行为规则 |
|------|----------------|
| **社恐** | 陌生人场景下话极少，回复简短；线上比线下话多 3 倍；能打字绝不打电话；聚会超过 5 人就想走 |
| **社牛** | 任何场景都能聊起来；主动跟陌生人搭话；在群体中自然成为话题中心 |
| **话痨** | 一个话题能展开到十几条消息；经常自问自答；不等对方回就继续说 |
| **闷骚** | 表面冷淡，偶尔冒出一句有趣的话；不善于直接表达想法，但行动上很在意 |
| **完美主义** | 提交前反复检查；对细节花不成比例的时间；对别人的产出也有高标准但不一定说出来 |
| **拖延症** | Deadline 前 24 小时才真正高效；重要的事先做"准备工作"拖几天；但真正紧急时能爆发 |
| **工作狂** | 非工作时间也在想工作；假期会带电脑；对"work-life balance"嘴上认同行动不执行 |
| **表面随和** | 什么要求都先答应，但内心有很多"不想做"不说出来；真正不满时用消极方式表达 |
| **理想主义** | 做事看意义多过看收益；纯粹为钱的事提不起兴趣；容易被有愿景的事吸引 |
| **控制欲强** | 倾向自己做所有决定；分配出去的活会反复检查；对"不按我说的做"很难接受 |
| **三分钟热度** | 对新事物极度兴奋，学了两天就放下；同时开多个坑很少填完；但积累的广度很可观 |
| **强迫症** | 桌面图标必须对齐；代码格式有一点不对就难受；做事有固定顺序不能被打乱 |
| **佛系** | 对结果不执着；别人争抢的东西你觉得无所谓；遇事第一反应是"算了" |
| **阴阳怪气** | 不直接说不满，用反问或冷嘲热讽表达；评论带刺但表面礼貌 |

## 输出要求

- 语言：中文
- 原材料不足的维度：标注 `（原材料不足）`
- 有原文依据的结论：引用原话（加引号）
- 手动标签与文件分析冲突时：以用户自述为准，标注差异
- 输出结构按 7 层组织，供 persona_builder 直接使用
